# API  
### Sign up  
**/api/signup**  
POST  
**Request**  
{ "login":"...", "email":"...", "password":"..."}  
**Response**  
""  
**Коды возврата**  
* 200 - ОК
* 409 - пользователь уже зарегистрирован
* 403 - пользователь залогинен
